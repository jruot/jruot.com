I believe AI is the future of software development. These are the tools I've found most effective in my own workflow for boosting productivity and code quality.

- [Opencode](https://opencode.ai/) – My favourite coding CLI tool at the moment.
- [Hermes Agent](https://github.com/NousResearch/hermes-agent) – Autonomous coding agent. See [running multiple Hermes agents on one VM](http://localhost:1313/blog/running-multiple-hermes-agents-on-one-vm/).
- [Z.ai Code Subscription](https://z.ai) – Currently the best value LLM coding subscription. Use GLM 4.6 for coding. It's not the smartest model but with good prompting and planning it's sufficient for most of the tasks. Use my referral link for a discount: https://z.ai/subscribe?ic=E2ARZ5VRHU
- [Codex CLI](https://github.com/openai/codex) – One of the smartest coding tools out there. It's very limited though with OpenAI's subscription. I recommend this for complex tasks.
- [Gemini CLI](https://github.com/google-gemini/gemini-cli) – Good for planning, UI related tasks, writing
- [Context7 MCP](https://github.com/upstash/context7) – Connect this MCP to your CLI tool like Opencode to get up to date documentation for different frameworks and libraries
- [Playwright MCP](https://github.com/microsoft/playwright-mcp) – Control your browser from your AI CLI tool. Automate UI testing.
- [n8n](https://n8n.io/) – Perfect for connecting AI tools and automating many kinds of tasks
